# Output Format

本文件描述 `bilibili-sentiment-dashboard` 技能输出结果的格式规范。

## 整体结构

输出遵循 **结论 -> 证据 -> 建议** 三段式结构，确保回答简洁有逻辑。

```
结论：<一句话总结舆情态势>
证据：
- <情感分析数据>
- <弹幕分析数据>
- <用户画像数据>
- <互动指标数据>
建议：
1. <up主运营建议1>
2. <up主运营建议2>
...
```

## 核心返回字段

### sentiment（情感分析）

| 字段 | 类型 | 说明 |
|------|------|------|
| `positive` | number | 正面评论占比（0-100） |
| `neutral` | number | 中性评论占比（0-100） |
| `negative` | number | 负面评论占比（0-100） |
| `sentimentLabel` | string | 主导情感标签：`positive` / `neutral` / `negative` |

**解读规则：**

- 正面占比 >= 65%：舆情积极，社区氛围良好
- 正面占比 40-64%：舆情中性，需关注潜在节奏风险
- 正面占比 < 40%：舆情偏负面，需进行社区氛围维护

### 弹幕分析（B站特色）

| 字段 | 类型 | 说明 |
|------|------|------|
| `danmakuDensity` | number | 弹幕密度（条/分钟） |
| `danmakuSentiment` | object | 弹幕情感分布（与评论情感分析格式相同） |
| `peakTimeRange` | array | 弹幕密集时间段 [开始秒数, 结束秒数] |

**弹幕解读注意事项：**

- B站用户习惯"弹幕互动"而非"评论互动"，弹幕密度是内容质量的直接反映
- 弹幕中常见"阴阳怪气"和"玩梗"，需结合语境解读情感
- 弹幕峰值时间段反映用户认为的"名场面"或"高光时刻"

### userProfile（用户画像）

| 字段 | 类型 | 说明 |
|------|------|------|
| `ageDistribution` | object | 年龄段分布，键为年龄段，值为百分比 |
| `genderDistribution` | object | 性别分布 |
| `topKeywords` | array | 评论高热词列表（Top 10），重点关注B站特色词汇 |
| `regionDistribution` | object | 地域分布（前 5 名省份/城市） |

**年龄分段：** 18-25、26-35、36-45、46+

**B站高热词示例：** `#硬核` `#干货` `#涨知识` `#三连` `#名场面` `#爷青回`

### conversionPotential（转化潜力）

| 范围 | 等级 | 说明 |
|------|------|------|
| 80-100 | 极高 | 三连率和投币率高，非常适合内容变现 |
| 65-79 | 高 | 社区氛围好，up主商业化潜力大 |
| 45-64 | 中等 | 有一定粉丝基础，需优化变现路径 |
| 0-44 | 低 | 社区活跃度有限，建议先提升内容质量 |

### engagementMetrics（互动指标）

| 字段 | 类型 | 说明 |
|------|------|------|
| `likes` | number | 点赞数 |
| `comments` | number | 评论数 |
| `shares` | number | 分享/转发数 |
| `danmakuCount` | number | 弹幕总数（B站特色指标） |
| `tripleRate` | number | 三连率（点赞+收藏+投币的综合指标） |
| `coinRate` | number | 投币率 |

**B站特色衍生指标：**

- 三连率 = (点赞 + 收藏 + 投币) / 播放量 * 100%（衡量用户认可度的核心指标）
- 弹幕密度 = 弹幕总数 / 视频时长（条/分钟）
- 评论深度 = 评论平均字数（长评论代表高质量互动）

### suggestions（运营建议）

运营建议基于以下分析维度生成，聚焦B站平台和up主运营特性：

1. **内容系列化**：基于用户反馈和弹幕峰值，打造系列内容提升粉丝粘性
2. **三连引导**：在视频中合理引导三连，提升综合互动指标
3. **社区运营**：利用弹幕文化维护高质量社区氛围
4. **变现策略**：结合用户画像和互动指标，制定B站特有的创作激励/接广告策略

## B站平台特色

- **三连文化**：点赞+收藏+投币是B站特有的用户认可方式，三连率是up主实力的象征
- **弹幕互动**：弹幕是B站内容体验的重要组成部分，弹幕密度反映内容质量
- **社区氛围**：B站用户重视社区归属感，up主与粉丝的关系更像是"朋友"
- **长视频优势**：B站支持长视频，内容深度和干货密度是核心竞争力

## 输出示例

### 成功响应（B站视频）

```
结论：该视频用户反馈整体正面（正面评论占比 71%），弹幕活跃度高，社区氛围良好。

证据：
- 情感分析：正面 71%，中性 21%，负面 8%
- 弹幕密度：平均 120 条/分钟（同类均值 85 条）
- 评论峰值：视频 5:00-8:00（干货密集区）
- 转化潜力评分：82/100（高）

用户画像：
- 主要受众：18-30 岁男性（占比 65%）
- 高热词：#硬核 #干货 #涨知识 #三连
- 互动特点：投币率 12%，三连率 18%

建议：
1. 内容系列化：拆分为"入门篇/进阶篇/实战篇"
2. 视频中植入相关课程/工具推荐，引导转化
3. 建立粉丝群维护高质量用户关系
```

### 舆情中性响应（需语境解读）

```
结论：该视频舆情偏中性（中性评论占比偏高），弹幕中"阴阳怪气"和"玩梗"较多，需结合语境解读。

证据：
- 情感分析：正面 38%，中性 48%，负面 14%
- 弹幕密度：平均 95 条/分钟（中等水平）
- 评论区高热词：#离谱 #就这？ #下次还来 #经典
- 转化潜力评分：55/100（中等）

用户画像：
- 主要受众：18-25 岁男性（占比 58%）
- 互动特点：弹幕互动强，评论相对沉默

建议：
1. B站用户习惯"弹幕互动"而非"评论互动"，重视弹幕节奏把控
2. 对弹幕中的质疑声音进行正向引导，可在视频中正面回应
3. 利用"经典/名场面"弹幕文化，制作系列视频提升用户粘性
```
