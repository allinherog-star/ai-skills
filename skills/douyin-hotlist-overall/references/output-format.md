# Output Format

本文件描述 `douyin-hotlist-overall` 技能输出结果的格式规范。

## 整体结构

输出遵循 **结论 -> 证据 -> 建议** 三段式结构，确保回答简洁有逻辑。

```
结论：<一句话总结，当前景点情况>
证据：
- <热点话题1>  热度排名 #<N>  热度值 <X>
- <热点话题2>  热度排名 #<N>  热度值 <X>
- ...
建议：
1. <针对第1个话题的创作/跟进建议>
2. <针对第2个话题的创作/跟进建议>
3. ...
```

## 核心返回字段

API 实际返回的热点数据字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| `title` | 话题标题/标签 | `#某明星官宣恋情` |
| `rank` | 当前热度排名 | `1` |
| `hot_value` | 热度值（数值越大热度越高） | `99999999` |
| `desc` | 话题描述（可选） | `某明星微博官宣恋情引发全网热议` |
| `reason` | 上榜原因（可选） | `事件持续发酵中` |

## 字段说明

### 热度排名（rank）

话题在抖音热榜中的绝对排名位置。数字越小，热度越高。

- 排名前 5：超高热度，流量巨大但竞争激烈
- 排名 6-20：高热度，流量充沛
- 排名 21-50：上升中，有流量空间
- 排名 50+：新兴话题，先发优势明显

### 热度值（hot_value）

话题当前的绝对热度数值，反映话题的整体热度和讨论量。

- 热度值越高，说明话题越热门
- 不同话题的热度值差异可能很大，优先关注排名而非绝对数值

### 话题描述（desc）

对热搜话题的简要描述，帮助用户快速理解话题背景。

### 上榜原因（reason）

话题上榜的主要原因说明，帮助用户判断是否值得跟进。

## 输出示例

### 成功响应（结论 -> 证据 -> 建议）

```
结论：以下是抖音实时热搜榜 Top 10，娱乐和社会类话题占据主导。

证据：
- #某明星官宣恋情  热度排名 #1  热度值 9999万+  上榜原因：明星效应
- #某品牌联名新品  热度排名 #2  热度值 8500万  上榜原因：联名热度高
- #某影视剧名场面  热度排名 #3  热度值 7200万  上榜原因：剧情讨论度高
- #某社会热点事件  热度排名 #4  热度值 6800万  上榜原因：事件持续发酵
- #某生活小技巧  热度排名 #5  热度值 5500万  上榜原因：实用性强

建议：
1. #某明星官宣恋情：娱乐类内容可蹭热度，建议从情感分析角度切入
2. #某品牌联名新品：适合电商类内容，可做开箱或测评视频
3. #某影视剧名场面：适合影视剪辑类内容，可做二次创作或解读
```

### 无结果响应

```
结论：当前热搜榜单正在更新中，请稍后重试。

建议：
1. 稍后再试，抖音热搜更新频繁
2. 关注抖音热搜榜实时查看最新话题
3. 尝试其他热点类技能，如 douyin-realtime-hot-rise（上升热点）
```
