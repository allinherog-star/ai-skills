# Output Format

本文件描述 `xhs-sentiment-dashboard` 技能输出结果的格式规范。

## 整体结构

输出遵循 **结论 -> 证据 -> 建议** 三段式结构，确保回答简洁有逻辑。

```
结论：<一句话总结舆情态势>
证据：
- <情感分析数据>
- <用户画像数据>
- <互动指标数据>
建议：
1. <运营建议1>
2. <运营建议2>
...
```

## 核心返回字段

### sentiment（情感分析）

| 字段 | 类型 | 说明 |
|------|------|------|
| `positive` | number | 正面评论占比（0-100） |
| `neutral` | number | 中性评论占比（0-100） |
| `negative` | number | 负面评论占比（0-100） |
| `sentimentLabel` | string | 主导情感标签：`positive` / `neutral` / `negative` |

**解读规则：**

- 正面占比 >= 65%：舆情积极，种草效果良好
- 正面占比 45-64%：舆情中性，需关注用户反馈
- 正面占比 < 45%：舆情偏负面，需进行笔记优化和舆情引导

### userProfile（用户画像）

| 字段 | 类型 | 说明 |
|------|------|------|
| `ageDistribution` | object | 年龄段分布，键为年龄段，值为百分比 |
| `genderDistribution` | object | 性别分布 |
| `topKeywords` | array | 评论高热词列表（Top 10），重点关注小红书种草词 |
| `regionDistribution` | object | 地域分布（前 5 名省份/城市） |

**年龄分段：** 18-24、25-35、36-45、46+

**小红书高热词示例：** `#干货` `#种草` `#已入手` `#平替` `#氛围感`

### conversionPotential（转化潜力）

| 范围 | 等级 | 说明 |
|------|------|------|
| 80-100 | 极高 | 种草效果极佳，用户购买意向强烈 |
| 65-79 | 高 | 收藏率和互动率高，转化潜力大 |
| 45-64 | 中等 | 有一定种草效果，可优化转化路径 |
| 0-44 | 低 | 种草效果弱，建议优化笔记内容 |

### engagementMetrics（互动指标）

| 字段 | 类型 | 说明 |
|------|------|------|
| `likes` | number | 点赞数 |
| `comments` | number | 评论数 |
| `shares` | number | 分享数 |
| `collects` | number | 收藏数（小红书核心转化指标） |

**小红书特色衍生指标：**

- 收藏率 = 收藏数 / 点赞数 * 100%（小红书最核心的种草指标）
- 评论率 = 评论数 / 点赞数 * 100%（衡量用户深度互动意愿）
- 互动率 = (点赞 + 评论 + 收藏 + 分享) / 阅读量 * 100%

### suggestions（运营建议）

运营建议基于以下分析维度生成，聚焦小红书平台特性：

1. **笔记优化**：基于用户反馈和评论热点，提供内容改进方向
2. **种草引导**：利用高热词和收藏引导策略，提升种草效果
3. **KOC运营**：基于用户画像，制定精细化KOC合作策略
4. **转化策略**：结合收藏率和用户画像，制定小红书电商转化路径

## 小红书平台特色

- **收藏为王**：收藏数是衡量种草效果的核心指标，高收藏代表强购买意向
- **种草氛围**：小红书用户习惯"看完收藏以后买"，建议中应包含收藏引导策略
- **KOC文化**：普通用户（KOC）的真实分享比KOL更具说服力
- **女性主导**：用户以年轻女性为主，内容风格和话术需适配目标群体

## 输出示例

### 成功响应（小红书笔记）

```
结论：该笔记整体反响良好，正面情绪占主导（正面评论占比 74%），收藏率较高，用户认可度高。

证据：
- 情感分析：正面 74%，中性 19%，负面 7%
- 收藏率 15.3%（高于同类笔记均值 8.2%）
- 评论区高热词：#干货 #种草 #已入手
- 互动率：点赞 2.3w，收藏 8,600，评论 1,200

用户画像：
- 主要受众：25-35 岁都市女性（占比 71%）
- 兴趣标签：护肤、美妆、生活方式
- 地域分布：上海（22%）、北京（18%）、成都（12%）

建议：
1. 趁热度发布笔记 Part2，延续"种草清单"系列
2. 在评论区置顶"粉丝专属折扣码"提升转化
3. 私信高互动用户建立 KOC 关系链
```

### 舆情偏负面响应

```
结论：该笔记舆情偏负面（负面评论占比 48%），负面反馈集中在产品效果和价格两个方面。

证据：
- 情感分析：正面 32%，中性 20%，负面 48%
- 负面评论集中在：产品效果（42%）、价格偏高（35%）
- 收藏率 6.1%（低于同类均值 8.2%）
- 转化潜力评分：40/100（有提升空间）

用户画像：
- 主要受众：20-30 岁女性（占比 68%）
- 高热词：#踩雷 #性价比 #等发货 #观望

建议：
1. 在评论区主动回应负面反馈，展示解决问题的诚意
2. 推出限时折扣或赠品活动，挽回价格敏感型用户
3. 私信高互动用户了解具体不满点，进行一对一沟通
```
