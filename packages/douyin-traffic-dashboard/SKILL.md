# 抖音平台流量在哪?

## Overview

抖音流量分配大盘技能，帮助创作者、运营者和商家了解当前抖音平台上各内容分类的流量分布情况。通过调用 TikHub 抖音热榜上升数据，按 `traffic-distribution` 聚合模式呈现流量流向，让用户清晰看到哪些赛道正在获得平台流量倾斜，从而做出更明智的内容决策。

## Parameters

本技能 **无输入参数**（`formSchema` 为空对象 `{}`），直接调用 API 获取当前抖音流量分配大盘数据。

无需用户额外提供任何参数，直接执行即可。

## Execution

本技能使用 `execute` 调用模式。

1. 构造请求体：
```json
{
  "skillId": "douyin-traffic-dashboard",
  "params": {}
}
```

2. 发送 `POST` 请求到 `${AISKILLS_BASE_URL}/api/v1/execute`

3. 请求头需包含：
   - `Content-Type: application/json`
   - `X-API-Key: ${AISKILLS_API_KEY}`
   - `X-Tenant-Id: ${AISKILLS_TENANT_ID}`（默认 `default`）

4. 解析响应中的 `data.result.items`，按"结论 -> 证据 -> 建议"结构输出

完整 API 接口规范请参阅 `references/api-schema.md`。

## Output Format

输出遵循 **结论 -> 证据 -> 建议** 三段式结构。

```
结论：<一句话总结，当前抖音流量分配的核心发现>

证据：
- <分类1> 流量占比 <X>%，热度排名 #<N>，趋势 <上升/平稳/下降>
- <分类2> 流量占比 <X>%，热度排名 #<N>，趋势 <上升/平稳/下降>
- ...

建议：
1. <针对流量增长赛道的创作建议>
2. <针对新兴机会赛道的建议>
3. <针对高竞争赛道的差异化建议>
```

详细输出格式规范请参阅 `references/output-format.md`。

## Examples

### Example 1

**用户输入：** "抖音现在的流量都往哪些分类流？"

**技能输出：**
```
结论：当前抖音流量明显向美食、情感和生活方式三大分类集中，美食类流量占比持续扩大。

证据：
- 美食 流量占比约 28%，热度排名 #1，趋势 上升
- 情感 流量占比约 22%，热度排名 #2，趋势 平稳
- 生活方式 流量占比约 18%，热度排名 #3，趋势 上升
- 科技 流量占比约 12%，热度排名 #4，趋势 上升
- 娱乐 流量占比约 10%，热度排名 #5，趋势 下降

建议：
1. 美食类创作者可继续深耕，但建议寻找差异化角度（如地方特色、小众菜系）
2. 情感类内容流量稳定，适合情感类账号持续产出
3. 生活方式赛道正在获得流量倾斜，新账号可优先考虑入局
```

### Example 2

**用户输入：** "最近什么类型的短视频流量涨得最多？"

**技能输出：**
```
结论：近期"生活方式"和"科技"类内容的流量增长最为显著，是当前值得重点关注的上升赛道。

证据：
- 生活方式 流量占比 18%，热度排名 #3，趋势 快速上升（+15%）
- 科技 流量占比 12%，热度排名 #4，趋势 上升（+8%）
- 美食 流量占比 28%，热度排名 #1，趋势 平稳
- 时尚 流量占比 9%，热度排名 #6，趋势 上升（+5%）

建议：
1. 生活方式赛道处于上升期，建议尽快产出内容抢占先发优势
2. 科技类内容门槛较高，但竞争相对较小，适合有专业背景的创作者
3. 美食类虽流量最大但竞争激烈，若非头部账号建议另辟蹊径
```

### Example 3

**用户输入：** "抖音流量大盘怎么看？"

**技能输出：**
```

证据：
- 娱乐 流量占比约 30%，热度排名 #1，趋势 下降（-3%）
- 美食 流量占比约 25%，热度排名 #2，趋势 平稳
- 情感 流量占比约 15%，热度排名 #3，趋势 上升
- 教育 流量占比约 10%，热度排名 #5，趋势 上升

建议：
1. 娱乐类流量虽高但呈下降趋势，不建议新账号以此为主攻方向
2. 教育类正在获得平台流量扶持，有知识储备的创作者可重点投入
3. 综合来看，生活方式+垂直细分是当前最具潜力的流量切入点
```
