# Output Format

本文件描述 `douyin-sentiment-dashboard` 技能输出结果的格式规范。

## 整体结构

输出遵循 **结论 -> 证据 -> 建议** 三段式结构，确保回答简洁有逻辑。

```
结论：<一句话总结舆情态势>
证据：
- <情感分析数据>
- <用户画像数据>
- <互动指标数据>
建议：
1. <运营建议1>
2. <运营建议2>
...
```

## 核心返回字段

### sentiment（情感分析）

| 字段 | 类型 | 说明 |
|------|------|------|
| `positive` | number | 正面评论占比（0-100） |
| `neutral` | number | 中性评论占比（0-100） |
| `negative` | number | 负面评论占比（0-100） |
| `sentimentLabel` | string | 主导情感标签：`positive` / `neutral` / `negative` |

**解读规则：**

- 正面占比 >= 60%：舆情积极，适合正向放大
- 正面占比 40-59%：舆情中性，需关注负面风险
- 正面占比 < 40%：舆情偏负面，需立即进行危机公关

### userProfile（用户画像）

| 字段 | 类型 | 说明 |
|------|------|------|
| `ageDistribution` | object | 年龄段分布，键为年龄段，值为百分比 |
| `genderDistribution` | object | 性别分布 |
| `topKeywords` | array | 评论高热词列表（Top 10），重点关注抖音标签词 |
| `regionDistribution` | object | 地域分布（前 5 名省份/城市） |

**年龄分段：** 18-25、26-35、36-45、46+

**抖音高热词示例：** `#春季穿搭` `#平价好物` `#学生党` `#种草` `#OMG`

### conversionPotential（转化潜力）

| 范围 | 等级 | 说明 |
|------|------|------|
| 80-100 | 极高 | 非常适合引导转化，可植入强转化内容 |
| 60-79 | 高 | 转化潜力良好，可适当植入转化元素 |
| 40-59 | 中等 | 转化空间有限，建议优化内容策略 |
| 0-39 | 低 | 转化难度大，优先提升内容质量 |

### engagementMetrics（互动指标）

| 字段 | 类型 | 说明 |
|------|------|------|
| `likes` | number | 点赞数 |
| `comments` | number | 评论数 |
| `shares` | number | 转发/分享数 |
| `collects` | number | 收藏数 |

**抖音特色衍生指标：**

- 完播率 = 完整播放次数 / 播放量 * 100%（抖音核心指标）
- 互动率 = (点赞 + 评论 + 转发 + 收藏) / 播放量 * 100%
- 点赞关注比 = 点赞数 / 新增关注数（衡量账号吸引力）

### suggestions（运营建议）

运营建议基于以下分析维度生成，聚焦抖音平台特性：

1. **算法优化**：根据情感分析结果，调整内容标签和发布时机以获得更好推荐
2. **短视频节奏**：基于评论峰值时间，优化视频开头黄金3秒和内容节奏
3. **带货转化**：结合转化潜力评分和用户画像，制定抖音带货转化路径
4. **评论区运营**：利用高热词在评论区置顶引导，提升互动和转化

## 抖音平台特色

- **算法推荐机制**：抖音去中心化流量池，理解内容标签与用户兴趣匹配至关重要
- **完播率优先**：完播率是抖音核心指标，建议建议中包含内容时长优化建议
- **黄金3秒**：视频开头3秒决定用户是否继续观看
- **标签运营**：合理使用热门标签可获得额外推荐流量

## 输出示例

### 成功响应（抖音视频）

```
结论：该视频整体舆情偏正面（正面评论占比 68%），用户互动积极，但负面评论集中在视频后半段。

证据：
- 情感分析：正面 68%，中性 22%，负面 10%
- 评论峰值出现在视频 0-30 秒，正面情绪占比高达 75%
- 负面评论多在 2:15 处（涉及争议性话题）
- 转化潜力评分：78/100（中等偏高）

用户画像：
- 主要受众：18-25 岁女性（占比 62%）
- 高热词：#春季穿搭 #平价好物 #学生党
- 地域分布：广东（18%）、浙江（14%）、江苏（11%）

建议：
1. 视频后半段加入正向引导，降低负面情绪扩散风险
2. 结合"学生党平价好物"标签制作后续内容，延续热度
3. 在评论区置顶"学生党专属优惠"引导评论互动
```

### 舆情偏负面响应

```
结论：该视频舆情偏负面（负面评论占比 52%），需要立即进行舆情引导和危机公关。

证据：
- 情感分析：正面 28%，中性 20%，负面 52%
- 负面评论主要集中在：产品质量（38%）、物流速度（34%）、客服态度（28%）
- 转化潜力评分：35/100（低）

用户画像：
- 主要受众：25-35 岁女性（占比 65%）
- 高热词：#失望 #退货 #质量差 #物流慢
- 地域分布：山东（21%）、河南（16%）、河北（14%）

建议：
1. 立即发布公开声明回应负面关切，展示解决问题的态度
2. 为受影响用户提供补偿方案（退款/优惠券）
3. 在评论区置顶解决方案，降低后续负面扩散
4. 复盘产品/服务问题，避免类似问题再次发生
```
