# Output Format

本文件描述 `unified-comment-analysis` 技能输出结果的格式规范。

## 整体结构

输出遵循 **结论 -> 证据 -> 建议** 三段式结构，确保回答简洁有逻辑。

```
结论：<一句话总结舆情态势>
证据：
- <情感分析数据>
- <用户画像数据>
- <互动指标数据>
建议：
1. <运营建议1>
2. <运营建议2>
...
```

## 核心返回字段

### sentiment（情感分析）

| 字段 | 类型 | 说明 |
|------|------|------|
| `positive` | number | 正面评论占比（0-100） |
| `neutral` | number | 中性评论占比（0-100） |
| `negative` | number | 负面评论占比（0-100） |
| `sentimentLabel` | string | 主导情感标签：`positive` / `neutral` / `negative` |

**解读规则：**

- 正面占比 >= 60%：舆情积极，适合正向放大
- 正面占比 40-59%：舆情中性，需关注负面风险
- 正面占比 < 40%：舆情偏负面，需立即进行危机公关

### userProfile（用户画像）

| 字段 | 类型 | 说明 |
|------|------|------|
| `ageDistribution` | object | 年龄段分布，键为年龄段，值为百分比 |
| `genderDistribution` | object | 性别分布 |
| `topKeywords` | array | 评论高热词列表（Top 10） |
| `regionDistribution` | object | 地域分布（前 5 名省份/城市） |

**年龄分段：** 18-25、26-35、36-45、46+

**高热词示例：** `#干货` `#种草` `#已入手` `#学生党` `#平价好物`

### conversionPotential（转化潜力）

| 范围 | 等级 | 说明 |
|------|------|------|
| 80-100 | 极高 | 非常适合引导转化，可植入强转化内容 |
| 60-79 | 高 | 转化潜力良好，可适当植入转化元素 |
| 40-59 | 中等 | 转化空间有限，建议优化内容策略 |
| 0-39 | 低 | 转化难度大，优先提升内容质量 |

### engagementMetrics（互动指标）

| 字段 | 类型 | 说明 |
|------|------|------|
| `likes` | number | 点赞数 |
| `comments` | number | 评论数 |
| `shares` | number | 转发/分享数 |
| `collects` | number | 收藏数 |

**衍生指标：**

- 互动率 = (点赞 + 评论 + 转发 + 收藏) / 播放量 * 100%
- 收藏率 = 收藏数 / 点赞数 * 100%
- 评论率 = 评论数 / 播放量 * 100%

### suggestions（运营建议）

运营建议基于以下分析维度生成：

1. **舆情引导**：根据情感分析结果，给出引导正面舆论的策略
2. **内容优化**：基于用户反馈和评论热点，提供内容改进方向
3. **转化策略**：结合转化潜力评分和用户画像，制定转化路径
4. **用户运营**：基于用户画像，制定精细化运营策略

## 字段说明

### 情感分析（sentiment）

情感分析基于评论文本语义识别，计算正面、中性、负面评论的比例。

- **正面评论特征**：表达喜欢、认可、推荐、购买的评论
- **负面评论特征**：表达不满、抱怨、投诉、退货意向的评论
- **中性评论特征**：询问信息、无明显情感倾向的评论

### 用户画像（userProfile）

用户画像基于评论用户群体特征分析：

- **年龄分布**：反映内容受众定位，可用于投放策略
- **性别分布**：影响内容风格和营销话术
- **高热词**：反映用户关注点和兴趣点，指导内容选题
- **地域分布**：反映核心用户来源，可用于区域化运营

### 转化潜力（conversionPotential）

转化潜力评分综合以下因素计算：

1. 正面评论占比（权重 30%）
2. 收藏率（权重 25%）
3. 评论互动深度（权重 20%）
4. 用户购买意向评论占比（权重 15%）
5. 内容类型与商品的契合度（权重 10%）

### 互动指标（engagementMetrics）

互动指标是评估内容质量和用户参与度的重要依据：

- **点赞数**：内容受欢迎程度的基础指标
- **评论数**：用户参与度的重要指标，高评论代表话题性强
- **转发数**：内容传播能力，反映内容价值
- **收藏数**：用户未来转化意向的重要指标，高收藏代表强种草效果

## 输出示例

### 成功响应（抖音视频）

```
结论：该视频整体舆情偏正面（正面评论占比 68%），用户互动积极，但负面评论集中在视频后半段。

证据：
- 情感分析：正面 68%，中性 22%，负面 10%
- 评论峰值出现在视频 0-30 秒，正面情绪占比高达 75%
- 负面评论多在 2:15 处（涉及争议性话题）
- 转化潜力评分：78/100（中等偏高）

用户画像：
- 主要受众：18-25 岁女性（占比 62%）
- 高热词：#春季穿搭 #平价好物 #学生党
- 地域分布：广东（18%）、浙江（14%）、江苏（11%）

建议：
1. 视频后半段加入正向引导，降低负面情绪扩散风险
2. 结合"学生党平价好物"标签制作后续内容，延续热度
3. 在评论区置顶"学生党专属优惠"引导评论互动
```

### 成功响应（小红书笔记）

```
结论：该笔记整体反响良好，正面情绪占主导（正面评论占比 74%），收藏率较高，用户认可度高。

证据：
- 情感分析：正面 74%，中性 19%，负面 7%
- 收藏率 15.3%（高于同类笔记均值 8.2%）
- 评论区高热词：#干货 #种草 #已入手
- 互动率：点赞 2.3w，收藏 8,600，评论 1,200

用户画像：
- 主要受众：25-35 岁都市女性（占比 71%）
- 兴趣标签：护肤、美妆、生活方式
- 地域分布：上海（22%）、北京（18%）、成都（12%）

建议：
1. 趁热度发布笔记 Part2，延续"种草清单"系列
2. 在评论区置顶"粉丝专属折扣码"提升转化
3. 私信高互动用户建立 KOC 关系链
```

### 舆情偏负面响应

```
结论：该视频舆情偏负面（负面评论占比 52%），需要立即进行舆情引导和危机公关。

证据：
- 情感分析：正面 28%，中性 20%，负面 52%
- 负面评论主要集中在：产品质量（38%）、物流速度（34%）、客服态度（28%）
- 转化潜力评分：35/100（低）

用户画像：
- 主要受众：25-35 岁女性（占比 65%）
- 高热词：#失望 #退货 #质量差 #物流慢
- 地域分布：山东（21%）、河南（16%）、河北（14%）

建议：
1. 立即发布公开声明回应负面关切，展示解决问题的态度
2. 为受影响用户提供补偿方案（退款/优惠券）
3. 在评论区置顶解决方案，降低后续负面扩散
4. 复盘产品/服务问题，避免类似问题再次发生
```

### 无数据响应

```
结论：无法获取该内容的评论数据，可能原因：内容不存在、内容已下架或接口超时。

建议：
1. 确认链接是否正确，内容是否可访问
2. 稍后重试，部分平台数据有延迟
3. 尝试提供其他平台的分享链接
```
