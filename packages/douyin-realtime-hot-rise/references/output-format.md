# Output Format

本文件描述 `douyin-realtime-hot-rise` 技能输出结果的格式规范。

## 整体结构

输出遵循 **结论 -> 证据 -> 建议** 三段式结构，确保回答简洁有逻辑。

```
结论：<一句话总结，选题方向>
证据：
- <热点话题1> 热度排名 #<N>，排名<上升/下降> <X> 位
- <热点话题2> 热度排名 #<N>，排名<上升/下降> <X> 位
- ...
建议：
1. <针对第1个话题的创作建议>
2. <针对第2个话题的创作建议>
3. ...
```

## 核心返回字段

API 实际返回的热点数据字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| `title` | 话题标题/标签 | `#春季护肤` |
| `rank` | 当前热度排名 | `12` |
| `rank_diff` | 排名变化幅度（正数表示上升，负数表示下降） | `8` |
| `tag` | 话题所属分类 | `美妆` |
| `hot_value` | 热度值（可选） | `95234` |
| `desc` | 话题描述（可选） | `春季护肤误区大盘点` |

## 字段说明

### 热度排名（rank）

话题在抖音热榜中的绝对排名位置。数字越小，热度越高。

- 排名前 10：超高热度，流量巨大但竞争激烈
- 排名 11-30：高热度，流量充沛
- 排名 31-50：上升中，有流量空间
- 排名 50+：新兴话题，先发优势明显

### 变化幅度（rank_diff）

相比上一统计周期的排名变化，正数表示排名上升，负数表示排名下降。

- `+10` 及以上：爆发期切入最佳，但热度可能已接近峰值
- `+5` 至 `+9`：上升期，流量增长稳定，是推荐切入时机
- `+1` 至 `+4`：缓慢上升，可作为备选选题
- 负数或 0：热度衰退，不建议作为主攻方向

### 话题分类（tag）

话题所属的垂直领域分类，如美妆、美食、科技、娱乐等。用于结合用户的创作领域进行过滤。

### 选题建议

根据热点数据，为用户生成的个性化内容创作建议，应包含：

1. **切入角度**：从哪个角度切入话题更容易获得流量
2. **差异化方向**：如何在同类内容中做出差异化
3. **执行优先级**：哪些话题值得优先跟进

## 输出示例

### 成功响应（结论 -> 证据 -> 建议）

```
结论：近期"春季护肤"类话题流量显著上升，建议美妆类创作者优先跟进。

证据：
- #春季护肤 热度排名 #12，排名上升 +8 位
- #平价替代 热度排名 #25，排名上升 +15 位
- #成分党护肤 热度排名 #38，排名上升 +22 位

建议：
1. 立即拍摄"春季护肤误区"短视频，正面切入#春季护肤
2. 制作"大牌平替"对比视频，挂载购物车引流
3. 发布"成分党必看"科普内容，建立专业人设
```

### 无结果响应

```
结论：当前没有找到符合条件的上升热点话题。

建议：
1. 扩大筛选范围（尝试移除 keyword 或更改 tag）
2. 稍后再试，热点话题更新频繁
3. 关注抖音热搜榜实时查看最新话题
```
