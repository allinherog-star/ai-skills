# Output Format

本文件描述 `kuaishou-sentiment-dashboard` 技能输出结果的格式规范。

## 整体结构

输出遵循 **结论 -> 证据 -> 建议** 三段式结构，确保回答简洁有逻辑。

```
结论：<一句话总结舆情态势>
证据：
- <情感分析数据>
- <用户画像数据>
- <互动指标数据>
建议：
1. <运营建议1>
2. <运营建议2>
...
```

## 核心返回字段

### sentiment（情感分析）

| 字段 | 类型 | 说明 |
|------|------|------|
| `positive` | number | 正面评论占比（0-100） |
| `neutral` | number | 中性评论占比（0-100） |
| `negative` | number | 负面评论占比（0-100） |
| `sentimentLabel` | string | 主导情感标签：`positive` / `neutral` / `negative` |

**解读规则：**

- 正面占比 >= 60%：舆情积极，老铁互动良好
- 正面占比 40-59%：舆情中性，需关注负面风险
- 正面占比 < 40%：舆情偏负面，需立即进行老铁关系维护

### userProfile（用户画像）

| 字段 | 类型 | 说明 |
|------|------|------|
| `ageDistribution` | object | 年龄段分布，键为年龄段，值为百分比 |
| `genderDistribution` | object | 性别分布 |
| `topKeywords` | array | 评论高热词列表（Top 10），重点关注快手老铁文化词汇 |
| `regionDistribution` | object | 地域分布（前 5 名省份/城市，快手下沉市场特色明显） |
| `priceSensitivity` | string | 价格敏感度：`高` / `中` / `低` |

**年龄分段：** 18-25、26-35、36-45、46+

**快手高热词示例：** `#老铁` `#666` `#支持` `#下单了` `#真实` `#接地气`

### conversionPotential（转化潜力）

| 范围 | 等级 | 说明 |
|------|------|------|
| 80-100 | 极高 | 老铁购买意向强烈，非常适合直播带货 |
| 65-79 | 高 | 私域流量活跃，转化潜力良好 |
| 45-64 | 中等 | 有一定老铁基础，需优化转化路径 |
| 0-44 | 低 | 老铁关系较弱，建议先加强粉丝维护 |

### engagementMetrics（互动指标）

| 字段 | 类型 | 说明 |
|------|------|------|
| `likes` | number | 点赞数 |
| `comments` | number | 评论数 |
| `shares` | number | 分享数 |
| `followers` | number | 新增关注数（快手私域关键指标） |
| `liveViewers` | number | 直播峰值在线人数（直播场景） |

**快手特色衍生指标：**

- 老铁互动率 = 评论数 / 粉丝数 * 100%（衡量老铁粘性的核心指标）
- 关注转化率 = 新增关注数 / 播放量 * 100%（衡量内容吸引力的指标）
- 直播互动率 = 评论数 / 峰值在线人数 * 100%（直播带货核心指标）

### suggestions（运营建议）

运营建议基于以下分析维度生成，聚焦快手平台和"老铁文化"特性：

1. **老铁关系维护**：基于情感分析结果，制定老铁关系维护策略
2. **下沉市场策略**：结合用户画像和价格敏感度，制定下沉市场转化路径
3. **直播运营**：利用直播弹幕互动，提升直播带货转化率
4. **私域转化**：基于快手特有的私域流量优势，制定老铁电商转化策略

## 快手平台特色

- **老铁文化**：快手用户之间以"老铁"相称，强调真实、接地气的内容和真诚互动
- **下沉市场**：快手用户主要来自三四五线城市和农村，价格敏感度高
- **私域流量**：快手重视私域流量运营，粉丝粘性是核心竞争力
- **直播带货**：快手电商以直播带货为核心场景，老铁信任是转化的关键

## 用户画像特色

快手用户画像有以下下沉市场特点：

| 特点 | 说明 |
|------|------|
| 年龄偏大 | 主要受众为 25-45 岁，比抖音用户群体年龄偏大 |
| 地域下沉 | 用户主要分布在三四五线城市及农村地区 |
| 价格敏感 | 对价格敏感度高，追求性价比 |
| 熟人社交 | 社交关系偏向熟人，信任感更强 |

## 输出示例

### 成功响应（快手视频）

```
结论：该视频评论以正面为主（正面占比 62%），但负面反馈需关注，互动率整体中等。

证据：
- 情感分析：正面 62%，中性 24%，负面 14%
- 负面评论集中在"价格偏高"和"物流速度"方面
- 转化潜力评分：65/100（有提升空间）

用户画像：
- 主要受众：25-45 岁下沉市场用户（占比 78%）
- 价格敏感度：高
- 地域分布：山东（19%）、河南（17%）、河北（15%）

建议：
1. 针对负面反馈（价格/物流）在评论区公开回应，展示服务态度
2. 推出限时优惠券或满减活动，提升下单转化率
3. 与粉丝建立更紧密互动（如直播连麦、粉丝见面会）
```

### 直播舆情响应

```
结论：本次直播整体舆情偏正面（正面占比 71%），老铁互动积极，但中后期节奏下滑。

证据：
- 情感分析：正面 71%，中性 19%，负面 10%
- 直播峰值在线人数 3.2 万，平均在线 1.8 万
- 老铁评论高热词：#老铁666 #下单了 #快点上 #支持
- 转化潜力评分：75/100（较高）

用户画像：
- 主要受众：25-40 岁下沉市场用户（占比 82%）
- 价格敏感度：高（偏好性价比产品）
- 地域分布：河北（21%）、山东（18%）、河南（15%）

建议：
1. 直播后半段保持互动节奏，避免老铁流失
2. 推出"老铁专属价"强化情感连接，提升下单率
3. 直播后发布切片视频，延续热度并引流至下次直播
```
