# Output Format

本文件描述 `douyin-kol-search` 技能输出结果的格式规范。

## 整体结构

输出遵循 **结论 -> 证据 -> 建议** 三段式结构，确保回答简洁有逻辑。

```
结论：<一句话总结，KOL 搜索的核心发现>

证据：
- @<博主昵称>  粉丝 <X>w  商业价值 <等级>  互动率 <X>%  <内容方向>

建议：
1. @<博主昵称>：<合作建议>
2. @<博主昵称>：<合作建议>
3. ...
```

## 核心返回字段

API 实际返回的 KOL 数据字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| `nickname` | 博主昵称 | `@零食控小美` |
| `follower_count` | 粉丝数量（单位：w 表示万） | `120w` |
| `commercial_value` | 商业价值等级（S/A/B/C） | `A` |
| `avg_engagement` | 平均互动率（%） | `8.5%` |
| `category` | 内容分类/赛道 | `零食`, `美妆` |
| `content_style` | 内容风格/方向 | `零食评测`, `美妆教程` |

## 字段说明

### 粉丝数量（follower_count）

博主在抖音平台的粉丝总数，反映博主的基础影响力。

- **100w 及以上**：头部博主，影响力大但合作门槛高
- **50w - 99w**：腰部博主，商业价值较高，性价比较好
- **10w - 49w**：成长型博主，互动率通常较高，适合初创品牌
- **10w 以下**：新锐博主，适合小预算测试或长尾合作

### 商业价值等级（commercial_value）

综合评估博主带货能力、粉丝质量和内容商业化潜力的等级评分。

- **S 级**：顶级带货博主，GMV 表现优异，适合品牌代言
- **A 级**：高商业价值博主，转化率较高，适合种草合作
- **B 级**：中等商业价值博主，适合中腰部品牌合作
- **C 级**：潜力博主，适合小预算测试或长期培养

### 互动率（avg_engagement）

粉丝对博主内容的互动程度，包括点赞、评论、收藏、分享等行为的综合比率。

- **8% 及以上**：高互动率，粉丝粘性强，商业转化潜力大
- **5% - 7.9%**：中等互动率，内容质量稳定
- **3% - 4.9%**：偏低互动率，可能存在粉丝水分或内容质量问题
- **3% 以下**：低互动率，需谨慎评估商业价值

### 内容分类（category）

博主主要的内容领域/赛道，如：美食、美妆、服饰、生活方式、科技等。

### 内容风格（content_style）

博主内容的具体风格或形式，如：开箱、评测、教程、合集、日常等，帮助判断合作适配度。

## 输出示例

### 成功响应（结论 -> 证据 -> 建议）

```
结论：以下是零食垂类 KOL 推荐列表，按商业价值综合排序，建议重点关注 A 级博主。

证据：
- @零食控小美    粉丝 120w  商业价值 A  互动率 8.5%  主做零食评测
- @好吃嘴二狗  粉丝 85w   商业价值 A  互动率 7.2%  主做零食开箱
- @零食研究所  粉丝 60w   商业价值 B+ 互动率 6.8%  主做零食测评

建议：
1. @零食控小美：粉丝量大、互动率高，适合品牌联名和深度种草合作
2. @好吃嘴二狗：开箱类内容带货能力强，适合单品促销合作
3. @零食研究所：内容专业度高，适合长期口碑营销和品牌建设
```

### 无结果响应

```
结论：未找到符合条件的 KOL，请尝试调整搜索词或内容分类。

建议：
1. 尝试更通用的关键词，如"零食"改为"食品"
2. 清除内容分类筛选，扩大搜索范围
3. 尝试相关细分赛道，如"零食"可尝试"零食评测"、"办公室零食"等
```

### 错误响应

```
结论：KOL 搜索接口暂时不可用，请稍后重试。

建议：
1. 检查 AISKILLS_API_KEY 和 AISKILLS_TENANT_ID 环境变量是否正确配置
2. 稍后再试，API 可能存在临时性抖动
3. 如持续失败，可尝试其他技能如 douyin-hotlist-overall 查看热门内容趋势
```
